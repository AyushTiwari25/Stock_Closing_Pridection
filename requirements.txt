### **requirements.txt**

```plaintext
# Core Libraries
numpy==1.21.6
pandas==1.3.5

# Visualization Libraries
matplotlib==3.4.3
seaborn==0.11.2
plotly==5.3.1

# Machine Learning Libraries
scikit-learn==1.0.2
tensorflow==2.7.0
keras==2.7.0

# Utilities
jupyter==1.0.0
notebook==6.4.5
```

---

### **Explanation**  
- **Core Libraries**:  
  - `numpy` and `pandas` are used for data manipulation and numerical operations.  

- **Visualization Libraries**:  
  - `matplotlib`, `seaborn`, and `plotly` are used for creating static and interactive plots.  

- **Machine Learning Libraries**:  
  - `scikit-learn` for traditional ML models.  
  - `tensorflow` and `keras` for implementing and training the LSTM model.  

- **Utilities**:  
  - `jupyter` and `notebook` are included to run and visualize the Jupyter Notebooks.  

---

### Installation Command  
To install the dependencies, run:  
```bash
pip install -r requirements.txt
```
